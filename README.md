# USAGE
## full bundle
```
import WrVue from './index'

Vue.use(WrVue)
```
## as plugin
```
import WrInput from './components/WrInput'

Vue.use(WrInput)
```
## as component
```
import { WrInput } from './components/WrInput'

Vue.component('wr-input', WrInput)
```
